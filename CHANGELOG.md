# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024

### Added
- Initial release of Hostel Room Allocation System
- Student registration endpoint
- Room creation and management endpoints
- Room allocation functionality
- Admin dashboard routes
- Jest test suite with 82.35% coverage
- ESLint configuration for code quality
- Docker and Docker Compose support
- GitHub Actions CI/CD pipeline
- Comprehensive documentation

### Fixed
- ESLint console statement warnings
- CodeQL Action deprecation (updated to v3)
- Database connection error handling

### Security
- CodeQL security scanning configured
- Input validation on API endpoints
- Environment variable handling
